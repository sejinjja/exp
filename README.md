# exp

Minimal Node.js project scaffold.

## Project Structure

- `src/`: application source code
- `tests/`: tests using `node --test`
- `docs/`: documentation
- `.github/workflows/ci.yml`: CI pipeline

## Quick Start

1. Install dependencies:
   `npm.cmd install`
2. Run quality checks:
   `npm.cmd run check`
3. Run tests:
   `npm.cmd test`
