# Support

## Getting Help

Use the following channels based on your request type:

1. Bug reports: open a GitHub issue using the bug template
2. Feature requests: open a GitHub issue using the feature template
3. Security vulnerabilities: follow `SECURITY.md` and report privately

## Response Expectations

Maintainer response targets:

1. Initial response: within 3 business days
2. Triage/update: within 7 business days

## Scope

Support is provided on a best-effort basis for the latest `main` branch and
the latest tagged release.
